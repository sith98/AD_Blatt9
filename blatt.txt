O(n):
Berechne für alle i in [1, n] das Wert-pro-Volumen-Verhältnis r_i = w_i / v_i

O(n*log(n)):
Sortiere absteigend nach r.
(Gegenstände sind neu von 1 bis n durchnummeriert.)

O(n):
Setze alle a_i auf 0
Sei s(i) = Summe von v_k für k in [1, i]
Gehe alle Gegenstände von 1 bis n durch.
Setze a_i auf 1, solange W >= s(i).
Wenn s(i - 1) < W < s(i), setze a_i auf v_i / (W - s(i - 1)).
Breche ab.


Beispiel (Teilaufgabe 2):
w1 = 5      v1 = 4
w2 = 3      v2 = 3
w3 = 3      v3 = 3
